# FAQ Maslam - Q&A Permanen (Sumber RAG)

Sumber resmi: https://maslam.id/faq-maslam
Terakhir sinkron: 2026-04-16

## 1) Apa itu Maslam?
Maslam (Masajid Al Alam) adalah aplikasi ERP untuk pengelolaan masjid.

## 2) Apa saja fitur utama Maslam?
Mencakup data master masjid, kegiatan, jamaah, aset, inventory, ziswaf, keuangan, Idul Fitri, Idul Adha, manajemen user, dan pembaruan berkala.

## 3) Tipe masjid apa saja yang bisa menggunakan Maslam?
Semua tipe masjid dapat menggunakan Maslam.

## 4) Bagaimana cara mengakses Maslam?
Registrasi lembaga via Maslam DKM, menunggu approval dan password email, ikut grup, belajar mandiri, lalu pendampingan live.

## 5) Platform apa yang digunakan Maslam?
Web Apps dan Mobile Apps.

## 6) Apakah Maslam aman digunakan?
Ya. Menggunakan akses berbasis peran, program bug hunter, dan backup harian.

## 7) Penyimpanan data aplikasi Maslam
Data disimpan di cloud (bukan on-premise per masjid).

## 8) Kemampuan kustomisasi
Kustomisasi ada namun terbatas; fokus pada kebutuhan umum sebanyak mungkin masjid.

## 9) Apakah masjid lain bisa melihat data masjid saya?
Tidak. Hak akses tiap masjid terpisah.

## 10) Apakah pihak Maslam bisa melihat data masjid saya?
Pihak Maslam dapat melihat data untuk operasional, namun password pengguna dienkripsi satu arah.

## 11) Berapa biaya menggunakan aplikasi ini?
Aplikasi Maslam gratis. Tidak ada biaya penggunaan, termasuk pendampingan dan bantuan teknis.

## 12) Dari mana pendanaan Maslam berasal?
Dari komunitas CITCOM (C-Level IT Community) melalui kontribusi finansial dan dukungan sumber daya.
