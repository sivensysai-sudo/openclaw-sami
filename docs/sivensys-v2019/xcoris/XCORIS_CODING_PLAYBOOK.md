# XCORIS_CODING_PLAYBOOK

Playbook cepat sebelum coding lintas xcoris:

1. Tentukan domain & transaksi target.
2. Buka `XCORIS_CODING_IMPACT_MAP.md` untuk shortlist file.
3. Validasi status/workflow/jurnal di matriks domain terkait.
4. Edit minimal-impact (small patch), lalu test flow create->approve->cancel/delete.
5. Verifikasi query/update pada tabel header+detail+workflow+glt.
6. Dokumentasikan behavior change ke knowledge silo.
